Python使用技巧之基础篇

Posted on 2018-05-26 | In Python |

总结python使用过程中有用的函数和技巧。

基础知识

Number相关

因为python是动态语言，不需要指定变量的类型，如果想要指定类型，可以在赋值时进行操作。如： long型是无限大小整数，整数 最后一项是大写或小写的L

python常见类型转换

  1. int(); long(); float();
  2. str()
  3. repr() 将对象转换为表达式字符串
  4. tuple(); list();
  5. chr() 将一个int转换为字符（ASCII）
  6. unichr() 将一个整数转为字符（Unicode）
  7. ord() 将一个字符转换成对应的ASCII值
  8. hex() 将一个整数转换成十六进制字符串
  9. oct() 将一个整数转换成八进制字符串

    常见数学函数

  10. abs(); log(x, y); log10(); max(); min(); pow(); sqrt()
  11. ceil() 返回数字的上入整数，如math.ceil(4.1) 返回5
  12. exp(x) 返回e的x次幂
  13. floor(x) 返回数字的下舍整数，如math.floor(4.9) 返回4
  14. modf(x) 返回x的整数部分与小数部分，两部分的数值符号与x相同，整数部分以浮点型表示。

    随机数

  15. choose(seq) 从序列seq中随机选择一个元素返回
  16. randrange(start, stop, step) 从指定范围内，按指定基数递增的集合中获取一个随机数
  17. seed() 改变随机数种子
  18. shuffle(seq) 将序列所有元素洗牌
  19. uniform(x, y) 随机生成一个实数，取值范围在[x,y]之间

    内置常量

  20. pi 圆周率
  21. e 自然常数

字符串

Python 使用反斜杠()转义特殊字符，如果你不想让反斜杠发生转义，可以在字符串前面添加一个 r，表示原始字符串：

list

list.insert(index, obj)

常见函数

round()对变量进行四舍五入

在python3中，可以使用round()函数进行四舍五入输出：

round(x, n)

print函数

  1. value 表示要打印的值，可以为任意个参数
  2. sep 当要打印多个value时，各个value之间的分隔方式。默认值为空格
  3. end value打印结束后输出的符号，默认为回车换行，如果不希望打印后换行，可以将其修改为空字符串。
  4. file 设置输出设备，默认为输出到终端，可以这是file=文件存储对象，将内容打印到文件中。
  5. flush 刷新，默认为False，不刷新

编程技巧

数据向量化

在python中，对w和x进行乘法时，使用numpy的向量乘法要远快于使用for循环进行逐步相乘，一般情况下前者比后者快300倍左右。所以对ML代码尽可能向量化将大大加快运行速度。
经验法则：只要有其他可能，就不要使用显式for循环

python广播机制

python的广播机制可以让不同维度的向量或者矩阵进行标量运算，同时也允许矩阵和实数进行标量运算。
在不精通广播机制的情况下，python-numpy经常会导致一些奇怪的bug，以下为使用广播机制构建ML程序的建议：

  1. 不要使用秩为1的标量数组，而应该创建向量

这样会让程序更加容易理解
同时也可以用reshape将秩为1的数组转换为二维矩阵：’a.reshape(5, 1)’

  1. 在不确定数据的形状时，通常使用assert()声明，避免程序出现奇怪的bug

使用assert确保中间值

我们知道python是动态语言，不需要指定类型，方便的同时往往而已会让程序不可控，尤其是在数据处理相关的任务上。而使用assert语句可以在程序中对代码进行判断，大大加强代码的可读性。而且assert的运行速度非常快，几乎不会消耗运行时间。
