Python之Pandas

Posted on 2018-11-23 | In Python |

总结Pandas的常见用法。

Pandas

唯一值

  • 获取Series中的唯一值，返回对Series的set：series.unique()

  • 获取Series中唯一值的个数：series.nunique()

  • Series中每个元素出现次数

分位数

pd.quantile(num)用来显示分位数，例如num=0.25，表示将所有数据从小到大排序的第25%的数据，num=0.5时也就是中位数了。

pd.sort_value()

对pd进行排序，需要注意的是by不仅可以为一个column，也可以传入一个column list，先按照list中的第一个元素排序，如果第一个元素相同就按照第二个元素排序。

pd.groupby().agg()

对给定df的指定列属性进行group，然后显示每个group中的指定统计数值
