C++ vector容器

Posted on 2018-11-12 | In C++ |

向量（Vector）是一个封装了动态大小数组的顺序容器（Sequence Container）。跟任意其它类型容器一样，它能够存放各种类型的对象。可以简单的认为，向量是一个能够存放任意类型的动态数组。

特性

  1. 顺序序列
  2. 动态数组
  3. 动态感知内存分配器

基本函数

构造函数

  1. vector() v1;
  2. vector(int nSize) v2;
  3. vector(int nSize, const t& t) v3; 创建一个vector，元素个数为nSize,且值均为t
  4. vector(v3) v4; 用已经存在的vector创建新的vector。
  5. vector(v3.begin()+1, v3.begin()+4) v5;

    增加函数

  6. void push_back(const T& x) 尾部添加一个元素
  7. iterator insert(iterator it, const T& t) 迭代器指向元素前增加一个元素x
  8. iterator insert(iterator it, int n, const T& t) 向量中迭代器指向元素前增加n个相同的元素x

    删除函数

  9. iterator erase(iterator it):删除向量中迭代器指向元素
  10. iterator erase(iterator first,iterator last):删除向量中[first,last)中元素
  11. void pop_back():删除向量中最后一个元素
  12. void clear():清空向量中所有元素

    遍历函数

  • reference at(int pos):返回pos位置元素的引用
  • reference front():返回首元素的引用
  • reference back():返回尾元素的引用
  • iterator begin():返回向量头指针，指向第一个元素
  • iterator end():返回向量尾指针，指向向量最后一个元素的下一个位置
  • reverse_iterator rbegin():反向迭代器，指向最后一个元素
  • reverse_iterator rend():反向迭代器，指向第一个元素之前的位置

    判断函数

  • bool empty() 判断是否为空

    大小函数

  • int size() const:返回向量中元素的个数
  • int capacity() const:返回当前向量中所能容纳的最大元素值
  • int max_size() const:返回最大可允许的vector元素数量值

    其他函数

  • void assign(int n,const T& x):设置向量中第n个元素的值为x
  • void assign(const_iterator first,const_iterator last):向量中[first,last)中元素设置成当前向量元素

    algorithm中的辅助函数

    通过 ‘#include‘可以使用一些算法对vector中的元素进行操作

  • sort(vec.begin(), vec.end()) 对vector中元素排序。在用sort函数进行排序时，可以通过指定比较方式，实现复杂数据的排序或满足特定的排序内容：

  • reverse(vec.begin(), vec.end()) 对vector元素进行翻转

  • iterator max_element(v.begin(), v.end()) 返回vector中最大元素的位置的指针

  • iterator min_element(v.begin(), v.end())
