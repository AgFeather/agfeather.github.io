装饰器

装饰器类似于java中的面向切面编程，可以将各个函数中重复的操作（插入日志，性能测试，事务处理，缓存等）抽离出来，将这些于函数原本功能无关的雷同代码定义在装饰器中，概括的讲： 装饰器的作用就是为已经存在的对象添加额外的功能
